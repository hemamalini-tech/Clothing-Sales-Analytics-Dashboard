# Clothing Sales Analytics Dashboard

## Project Overview

This project analyzes clothing store sales data using Power BI, SQL, and Python. The dashboard provides business insights into sales performance, customer activity, product trends, and revenue distribution.

## Key Objectives

* Analyze total sales performance
* Identify top-selling products
* Compare revenue across product categories
* Evaluate city-wise revenue distribution
* Track key business KPIs

## Tools Used

* Power BI
* MySQL
* SQL
* Python
* Excel

## Dashboard Features

* KPI Cards (Revenue, Orders, Customers, Products)
* Revenue by Category Analysis
* Revenue by City Analysis
* Top Selling Products Visualization
* Interactive Category Filter
* Dynamic Dashboard Insights

## SQL Analysis

The SQL folder contains queries for:

* Total Revenue
* Total Orders
* Total Customers
* Revenue by Category
* Revenue by City
* Top Selling Products
